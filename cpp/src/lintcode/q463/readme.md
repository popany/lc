# [463 · 整数排序](https://www.lintcode.com/problem/463/)

- [463 · 整数排序](#463--整数排序)

难度: 入门

给一组整数，按照升序排序，使用选择排序，冒泡排序，插入排序或者任何 $O(n^{2})$ 的排序算法。

    样例
    样例  1:
        输入:  [3, 2, 1, 4, 5]
        输出:  [1, 2, 3, 4, 5]
        
        样例解释: 
        返回排序后的数组。

    样例 2:
        输入:  [1, 1, 2, 1, 1]
        输出:  [1, 1, 1, 1, 2]
        
        样例解释: 
        返回排好序的数组。
